# Farm-to-Table Platform

Farm-to-Table Platform is a web application that connects farmers and customers, allowing farmers to manage their items and customers to place orders directly.

## Features

- **Farmer Dashboard:**
  - Log in to your account as a farmer.
  - Add, edit items available for sale.
  - Manage inventory and update item details.

- **Customer Interface:**
  - Log in to your account as a customer.
  - Browse available items from local farmers.
  - Place orders for the desired products.

## Getting Started

### Prerequisites

- Node.js installed
- MongoDB database (local or cloud)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/farm-to-table-platform.git

1.Navigate to the project directory:

    cd farm-to-table-platform

2.Install dependencies:

    npm install

3.Set up environment variables:

    Create a .env file in the root directory.
    Add the following variables and update them with your MongoDB connection string and other details:
    env
    MONGODB_URI=your_mongodb_connection_string
    PORT=3000
5.Run the application:

    npm start
    Open your browser and visit http://localhost:8080 to access the application.
